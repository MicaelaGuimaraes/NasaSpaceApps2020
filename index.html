Projeto Cloud IO

A Cloud IO criou uma plataforma web para auxiliar pesquisadores e instituições governamentais, através da análise de imagens noturnas da Terra obtidas pelo satélite da NASA, trazendo métricas da poluição luminosa para nortear os impactos ambientais, demográficos e econômicos. Confira abaixo os pilares do nosso projeto:

Facíl compreensão

Uma das nossas principais missões é garantir uma transmissão de dados seguros e de fácil entendimento.

Unificação de dados

Plataforma capaz de armazenar diversos dados sem a necessidade de trafego externos.

Fonte de dados segura e confiável

A Cloud IO trás diversos dados para a plataforma de forma concreta e confiável, recebemos os dados via uma API da NASA.

Informações adicionais do projeto

  • O prinicipal problema

    Atualmente temos muitos governos e profissionais que não encontram uma plataforma completa e robusta para ter acesso a dados e determinados parametros de pesquisa, nossa equipe analisando essa situação se prôpos a desenvolver uma solução.

  • A nossa solução

    O nosso projeto tem com foco resolver o problema de falta de unificação de dados e analise e compreesão de dados com base em focos de luz nortuna capturadas diretamente por satélites da NASA e JAXA, sendo possível encontrar impactos ambientas, impactos ambientas regionais que causam efeitos negativos ou positivos em faunas e floras em áreas preservadas, áreas ruais e em áreas de dificil acesso para estudos e pesquisas.

  • Como nós vamos desenvolver a nossa solução

    A Cloud IO se prôpos a resolver o problema com tecnologias inovadoras e de última geração. Vamos trabalhar com uma plataforma web desenvolvida em Python com uma interface úsuario interativa e de fácil entendimento, preparamos um algoritmo capaz de analisar comparações de duas ou mais imagens e separa-las por pixels e fazer uma ánalise de componente para obter o resultado e uma taxa de poluição luminosa naquele área. Após obter esse resultado vamos passar esses dados e taxas para uma interface de úsuario que será capaz de retornar dados significativos de impacto ambiental, impacto governamental e impacto econômico.

  • O que se é esperado?

    Nós da Cloud IO esperamos que o projeto consiga atingir cada vez mais pessoas pelo mundo, esperamos que cada pessoas que utilizar o nossos serviços tenham uma expêriencia única e duradora. Vamos trabalhar cada vez mais para poder sempre adicionar mais funções e aprimoração diante das ferramentas que nós já temos.
